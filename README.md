# Superstore SQL Data Analysis

## Overview
This project analyzes the Superstore sales dataset (2011–2015) using SQL to uncover business insights about sales trends, profitability, shipping logistics, and customer segmentation across global markets and product categories.

## Problem Statement
The goal is to:
- Track and optimize sales and profit trends across regions, products, and time periods.
- Identify areas of low profitability, such as products or regions affected by discounts and shipping costs.
- Provide actionable recommendations for business strategy and customer targeting.

## Data Description
- **Size:** 51,290 rows, 24 columns
- **Categories:** Office Supplies, Furniture, Technology
- **Markets:** Africa, APAC, EMEA, LATAM, US, EU
- **Customer Segments:** Consumer, Corporate, Home Office

### Key Columns
- Order ID, Dates, Ship Mode
- Customer Info (ID, Name, Segment)
- Location (City, State, Country)
- Product Info (ID, Name, Category)
- Metrics: Sales, Quantity, Discount, Profit, Shipping Cost, etc.
